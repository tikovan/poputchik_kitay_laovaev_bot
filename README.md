
# Попутчик Китай | Laovaev.net

Готовый бот для поиска попутчиков и передачи посылок **из Китая и в Китай**.\n\nВажно: бот **не принимает оплату**. Пользователи сами договариваются между собой. Платформа выступает только как площадка для поиска.

## Что уже добавлено

- добавление объявлений двух типов:
  - ✈️ попутчик / поездка
  - 📦 посылка / запрос
- кнопки выбора стран
- поиск совпадений по маршруту
- список своих объявлений
- удаление объявления
- деактивация и повторная активация объявления
- поднятие объявления
- анонимная отправка сообщения владельцу через бота
- жалобы на объявления
- модерация объявлений
- базовая админка
- бан / разбан пользователя
- публикация активных объявлений в канал
- базовая антиспам-защита
- SQLite база

## Стек

- Python 3.11+
- aiogram 3
- SQLite

## Установка

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
```

Отредактируй `.env`

```env
BOT_TOKEN=YOUR_TELEGRAM_BOT_TOKEN
DB_PATH=bot.db
CHANNEL_USERNAME=@your_channel_username
ADMIN_IDS=123456789,987654321
BUMP_PRICE_TEXT=Площадка не принимает оплату. Пользователи договариваются между собой напрямую.
MAX_ACTIVE_POSTS_PER_USER=10
MIN_SECONDS_BETWEEN_ACTIONS=3
```

Запуск:

```bash
python bot.py
```

## Логика работы

### Пользователь
- запускает `/start`
- добавляет поездку или посылку
- выбирает страны кнопками
- вводит города, дату, вес, описание
- получает свое объявление
- смотрит совпадения
- отправляет сообщение владельцу через бота
- может удалить, деактивировать или поднять объявление

### Модерация
Если указаны `ADMIN_IDS`, новые объявления идут в статус `pending`.

Админы получают уведомление и могут:
- одобрить
- отклонить
- забанить автора объявления

### Канал
Если указан `CHANNEL_USERNAME`, активные объявления публикуются в Telegram-канал.

## Важные замечания

- Сейчас база данных — SQLite. Для продакшена лучше перейти на PostgreSQL.
- Публикации в канале в этой версии не редактируются и не удаляются автоматически при изменении объявления.
- Бот не принимает оплату. Все условия, доставка и расчеты обсуждаются пользователями напрямую.
- Для полноценного скрытого чата между сторонами можно добавить внутренний диалоговый режим.
